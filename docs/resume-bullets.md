# Resume Bullets

## 현대오토에버 AX 백엔드용

- Archive-Nexus에서 발생한 제조/모빌리티 운영 이벤트를 Outbox 패턴으로 발행하고, Archive-Ledger에서 고액·위험 비용 이벤트를 거래·원장·정산·승인 흐름으로 처리하도록 구현했습니다.
- ArchiveOS와 연동해 RAG 정책 근거, Human Approval Gate, DEGRADED 장애 격리를 구성하고 AI 기능 장애가 도메인 API로 전파되지 않도록 설계했습니다.

## 뱅크샐러드류 백엔드용

- 합성 거래 이벤트를 대상으로 idempotency key 기반 중복 방지, 거래 정규화, 원장 기록, 일별 정산 배치, 대사 기능을 구현했습니다.
- 중복 수신, 재처리, 승인 대기, 정산 제외, 불일치 집계 등 금융 데이터 백엔드에서 발생하는 운영 이슈를 테스트와 API로 검증했습니다.

## 현대카드류 직무용

- 카드/정비/구매성 합성 거래를 대상으로 승인 한도, 정책 근거, 원장 반영, 정산 배치, 감사 로그를 처리하는 금융성 백엔드 프로젝트를 구현했습니다.
- 합성 정책 문서를 RAG 근거로 연결하고, LLM 미설정 시에도 승인과 거래 처리 API가 유지되도록 fallback evidence와 DEGRADED 상태를 분리했습니다.
